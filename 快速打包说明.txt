    快速打包上传 GoDaddy 说明
====================================================================

方法 1: 使用 Windows 资源管理器手动打包
--------------------------------------------------------------------
1. 选择以下文件/文件夹:
   ☑ index.html
   ☑ styles.css
   ☑ script.js
   ☑ assets 文件夹 (整个文件夹)

2. 右键 -> 发送到 -> 压缩(zipped)文件夹
   或使用 WinRAR/7-Zip 压缩

3. 命名为: convergence-tech-deploy.zip

4. 上传到 GoDaddy:
   - 登录 GoDaddy cPanel
   - 打开 File Manager
   - 导航到 public_html
   - 上传 ZIP 文件
   - 解压缩


方法 2: 直接 FTP 上传 (推荐)
--------------------------------------------------------------------
1. 下载 FileZilla: https://filezilla-project.org/

2. 连接信息:
   Host: ftp.yourdomain.com
   Username: [您的 GoDaddy FTP 用户名]
   Password: [您的 FTP 密码]
   Port: 21

3. 上传以下文件到 public_html:
   - index.html
   - styles.css
   - script.js
   - assets/ (整个文件夹,保持结构)
